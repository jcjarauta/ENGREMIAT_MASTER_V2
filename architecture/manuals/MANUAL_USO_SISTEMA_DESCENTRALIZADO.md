# Manual de uso · Sistema descentralizado ENGREMIAT

## Regla principal
El sistema no se descentraliza copiando carpetas: se descentraliza asignando roles, contratos y protocolos.

## Flujo normal
1. MAIN_NODE coordina.
2. LEGACY_NODE entrega fuente o historico.
3. BACKUP_NODE guarda paquetes y evidencias.
4. Nuevos nodos se incorporan con plantilla.

## Estados de nodo
- PLANNED
- PREPARED
- READONLY_VALIDATED
- ACTIVE
- SUSPENDED
- LEGACY

## Politica
Un nodo sin contrato no ejecuta acciones reales.
