# ENGREMIAT_CONSTRUCTOR - Fase 006 - Automation by Gates

## Objetivo

Definir que puede automatizarse, que requiere humano y que queda bloqueado hasta autorizacion explicita.

## Permitido ahora

- Lectura local.
- Reportes locales.
- Simulacion local.
- Colas locales.
- Preparacion de drafts.

## Bloqueado

- IA externa.
- API externa.
- Worker real.
- Escritura real Control.
- Ejecucion real Prompter.
- Contacto cliente.
- Repo remoto.
- Browser.
- Clipboard.
