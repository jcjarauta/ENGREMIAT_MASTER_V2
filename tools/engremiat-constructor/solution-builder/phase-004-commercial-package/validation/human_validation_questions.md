# Human Validation Questions

- La necesidad es suficientemente concreta para una demo?
- La entidad objetivo pagaria por discovery, demo o piloto?
- La solucion encaja mejor como preventa, subvencion o partner?
- Que parte requiere humanos externos?
- Que metricas harian que el piloto sea aceptable?
- Que parte puede reutilizarse en otros sectores?
