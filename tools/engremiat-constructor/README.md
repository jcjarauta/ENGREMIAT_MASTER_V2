# ENGREMIAT_CONSTRUCTOR

ENGREMIAT_CONSTRUCTOR es la capa local que convertira necesidades detectadas por Engremiat en proyectos estructurados, documentados, versionables y potencialmente comercializables.

## Objetivo de la fase 001

Crear una fabrica local segura de proyectos desde necesidades normalizadas.

## Principios de seguridad

- No crea repositorios remotos en esta fase.
- No ejecuta APIs externas.
- No usa navegador, CDP ni clipboard.
- No activa workers reales.
- No hace push.
- Todo proyecto debe nacer con documentacion, estado, origen de demanda y trazabilidad.

## Flujo previsto

1. Necesidad detectada.
2. Necesidad normalizada.
3. Scoring de oportunidad.
4. Decision BUILD / ADAPT / HOLD / DISCARD.
5. Creacion de proyecto local.
6. Documentacion inicial.
7. Reporte para Engremiat Control.
8. Ficha Prompter para siguientes fases.
