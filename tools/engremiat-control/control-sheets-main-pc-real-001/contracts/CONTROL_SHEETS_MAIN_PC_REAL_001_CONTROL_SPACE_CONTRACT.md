# CONTROL_SHEETS_MAIN_PC_REAL_001 — Contrato del espacio de control humano

## Objetivo

Construir el primer espacio de control y supervision humana de Engremiat en Google Sheets, consolidando un modelo operativo comodo, verificable y reutilizable.

## Principio arquitectonico

Sheets sera la capa principal de control humano. Telegram vendra despues como capa conversacional y movil sobre el modelo ya consolidado.

## Funciones base

1. Estado del sistema.
2. Gates humanos.
3. Historial de runs.
4. Cola de tareas.
5. Diccionario operativo.
6. Manual humano.
7. Supervision de riesgos.
8. Puente futuro hacia Telegram.
